Operators:-
Python language supports several types of operators. In this section, we will focus on few of them.

Assignment Operators:-
Assignment operators are used to assign values to variables. 
Let us take = as an example. Equal sign in mathematics shows that two values are equal, however in Python it means we are storing a value in a certain variable and we call it assignment or a assigning value to a variable.
